## 🔧 发现并修复的问题

### 问题1: 记忆搜索返回格式错误
**错误**: `TypeError: unhashable type: 'slice'`  
**原因**: `_search()` 返回字典，但代码尝试切片  
**修复**: 修改返回格式为字符串  
**文件**: `evolution/tools/memory_tool.py`

### 问题2: 工具说明硬编码
**问题**: 工具使用说明写死在 `web_chat.py` 中  
**影响**: 工具更新时需要同步修改两处  
**修复**: 动态从工具实例的 `description` 字段读取  
**文件**: `evolution/chat/web_chat.py`

### 问题3: 时间上下文总是注入
**问题**: 每次对话都注入时间信息，浪费token  
**影响**: 降低LLM性能和响应速度  
**修复**: 实现 `needs_time_context()` 检测函数，按需注入  
**文件**: `evolution/chat/web_chat.py`

### 问题4: db工具action名称错误
**问题**: `web_chat.py` 使用 `query: SELECT...`，但工具不支持  
**影响**: 日程查询功能无法使用  
**修复**: 改为使用 `list_schedule` 等具体操作  
**文件**: `evolution/chat/web_chat.py`

### 问题5: intelligence工具action不匹配
**问题**: `web_chat.py` 调用 `fetch_latest`，但工具只支持 `briefing`  
**影响**: 情报收集功能无法使用  
**修复**: 统一使用 `briefing` action  
**文件**: `evolution/chat/web_chat.py`
